ASM Compiler/Disassembler Tool

Description

This tool provides two functions:

  1. Compile ASM files to binary or PE executables
  2. Disassemble executables back to ASM code

Requirements

  • Windows OS
  • NASM assembler (./nasm/nasm.exe)
  • NDISASM (./nasm/ndisasm.exe)
  • MinGW objdump (./mingw/objdump.exe)

Compiler Usage

compile.bat input.asm
  1. Select output format:
    • 1 - Flat binary (.bin)
    • 2 - Windows EXE (.exe)

Disassembler Usage

disasm.bat input.[exe|dll|bin]

Supported input formats:

  • .exe - PE executables
  • .dll - DLL files
  • .bin - Raw binary files

Disassembler Behavior

  • For .bin files: Uses ndisasm (32-bit mode)
  • For PE files: Uses objdump
  • Output: Creates inputname.asm file

Examples

Compilation:

compile.bat program.asm
[Select option 2 for EXE]

Disassembly:

disasm.bat program.exe
[Creates program.asm]

Troubleshooting

Common Errors:

  • Missing tools - Ensure all executables are in their correct folders
  • File not found - Check file paths and extensions
  • Disassembly fails - Verify file isn't corrupted or protected

Limitations

  • 32-bit architecture only
  • Basic PE file support
  • Output may need manual cleanup

Tool Locations

./nasm/nasm.exe      - Assembler
./nasm/ndisasm.exe   - Binary disassembler 
./mingw/objdump.exe  - PE disassembler
