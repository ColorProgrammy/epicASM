ASM Compiler/Disassembler Tool

Description

This batch script compiles ASM files to either flat binary (.bin) or Windows PE executable (.exe) format.

Requirements

  • Windows OS
  • NASM assembler (must be in ./nasm/nasm.exe)

Basic Usage

compile.bat input.asm

Operation Steps:

  1. Select output format:
    • 1 - Flat binary (.bin)
    • 2 - Windows EXE (.exe)
  2. The script checks for existing PE headers
  3. Generates temporary file with PE structure if needed
  4. Compiles using NASM

Examples

1. Compile to binary:

compile.bat program.asm
[Select option 1]

2. Compile to EXE (auto headers):

compile.bat program.asm
[Select option 2]

3. Compile to EXE (custom headers):

If your file already contains proper PE headers (MZ/PE signatures), they will be used as-is.

Customization

To add additional imported functions:

  1. Edit compile.bat
  2. Find the :build_pe section
  3. Add new functions following the existing pattern

Troubleshooting

"NASM not found" error

Ensure:

  • NASM is installed in ./nasm/
  • The executable is named nasm.exe

"Output file not created"

Possible causes:

  • Syntax errors in ASM code
  • Insufficient write permissions
  • Antivirus blocking EXE creation

EXE won't run

Try:

  • Running as administrator
  • Checking 32-bit compatibility
  • Verifying all required functions are imported

Limitations

  • 32-bit architecture only (x86)
  • Only .asm, .bin and dll files supported
  • Basic PE header generation
